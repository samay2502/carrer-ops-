# Resume Tailoring Skill

Claude skill for tailoring resumes to specific job descriptions.

## Features
- Bullet Quality Engine (scores and upgrades library bullets)
- ATS Simulation (scores resume before submission)
- Cover Letter Hook (generates matching cover letter from existing research)
- Multi-job batch mode
- Experience discovery interview

## Install
Place all files in your Claude skills directory:
~/.claude/skills/resume-tailoring/

The required file is SKILL.md. Supporting modules:
- ats-simulation.md
- bullet-quality-engine.md  
- cover-letter-hook.md
